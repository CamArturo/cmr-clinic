# Quick update notes

## Update `node_modules`

```
npm install
```

## Configure API Host

Adjust the API endpoint hostname in `nusxt.config.js`
```
  /*
  ** Axios module configuration
  */
  axios: {
    // See https://github.com/nuxt-community/axios-module#options
      baseURL: 'http://localhost:3001/'
  },
```
